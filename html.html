HTML Structure

HTML uses tags to define the structure of a web page...

        <!DOCTYPE html>
        <html>
          <head>
            <title>My Webpage</title>
          </head>
          <body>
            <h1>Welcome to my page!</h1>
            <p>This is some content.</p>
          </body>
        </html>
